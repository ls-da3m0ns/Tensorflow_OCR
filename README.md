# Tensorflow OCR
Perform OCR on alphabets and digits using tensorflow 

## Requirements
 * Tensorflow 2.2.0
 * Numpy      3.7.6
 * Opencv     
 * gdown
 * sklearn
 * Tensorboard

## Model 
We have Used ResNet styled architechture for better perfomance
